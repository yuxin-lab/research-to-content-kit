# Roadmap

This roadmap describes the planned development direction of Research to Content Kit.

The project is currently in early development. The first goal is to build a practical and reusable set of templates, examples, and lightweight tools for science communication workflows.

## v0.1.0

Goal: Build the initial project structure.

Planned items:

- Core README
- Template folder
- Example folder
- Script folder
- Documentation folder
- Research paper to video script template
- Medical content checklist template
- Slide outline template
- Acne science communication example
- Sugar and acne example
- Sleep and skin barrier example
- Simple Markdown outline generator

## v0.2.0

Goal: Improve content workflow usability.

Planned items:

- Add expert interview structuring template
- Add social post template
- Add bilingual Chinese-English template support
- Add more examples for dermatology, skincare, nutrition, and biotechnology topics
- Improve documentation for non-technical users
- Add contributor guide
- Add issue templates

## v0.3.0

Goal: Add lightweight automation and review tools.

Planned items:

- Add prompt templates for AI-assisted science communication
- Add compliance-aware wording review examples
- Add example input and output files
- Add basic tests for scripts
- Add release workflow
- Add more structured examples for public education content

## Long-term direction

The long-term goal is to make this project a practical open-source workflow toolkit for:

- Researchers explaining their work to the public
- Clinicians creating patient education materials
- Medical content teams building structured communication workflows
- Science communicators converting technical material into accessible content
- Students learning how to turn research into public education content

## Notes

This roadmap may change as the project develops.

Contributions, suggestions, and examples are welcome.
